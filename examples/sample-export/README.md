# Sample Export

This directory contains example manifest files (`manifest.json` and `manifest.csv`) that demonstrate the structure and fields produced by the exporter.

The actual audio files referenced in these manifests are **not included in this repository** because they are copyrighted material. Only the metadata records are kept as format examples.

The sample manifests still contain representative track metadata and cache-related identifiers. If you plan to redistribute this repository more broadly, review whether those sample rows should be further redacted for your use case.
